# KoeKey プライバシーポリシー

最終更新日: 2026-03-29

> [English version](/koekey/privacy-policy)

## 概要

KoeKey は音声をテキストに変換する音声入力アプリケーションです。本ポリシーでは、KoeKey がアクセス・収集・送信する個人情報の内容、その利用・保存・保護方法、およびお客様がご自身のデータに対して行使できる管理手段について説明します。

## 音声の処理方法

### ローカル処理（Whisper / sherpa-onnx）

ローカル音声認識モデル（Whisper または sherpa-onnx）を使用する場合、**すべての音声処理はお使いのデバイス上で完結します**。音声データがデバイスの外に出ることはありません。このモードでは KoeKey は音声データにアクセス・収集・送信しません。

### クラウド処理（Soniox, Deepgram）

クラウド音声認識プロバイダーの使用を選択した場合、音声は TLS で暗号化された接続を通じて選択したプロバイダーにストリーミング送信されます。音声は入力操作中のみ送信されます。

- **API キーを設定するまで、音声はクラウドプロバイダーに送信されません。** 有効なキーなしにはアプリは音声を送信できません。
- KoeKey は音声の録音を保存・保持しません。音声は入力セッション中のみメモリに保持され、終了後に破棄されます。
- お客様の音声データが KoeKey により **AI の学習に使用されることはありません**。

## 保存するデータ

### API キー

クラウドプロバイダーを使用する場合、ご自身の API キーを入力していただきます。API キーは OS のネイティブ資格情報ストア（Windows では Credential Manager、macOS では Keychain）に安全に保存されます。KoeKey は選択した音声認識プロバイダー以外のサーバーに API キーを送信しません。

### 文字起こし履歴

文字起こし履歴は AES-256-GCM で暗号化され、お使いのデバイスにローカル保存されます。暗号化キーは OS のセキュア資格情報ストアに保管されます。履歴データが外部サーバーに送信されることはありません。

### ファイル出力

文字起こし結果をファイルに保存する機能（ボイスメモ、リスニング出力など）を使用した場合、これらのファイルはお客様が選択した場所にプレーンテキストで保存されます。これらのファイルは暗号化されず、自動削除もされません。

### スクリーンショット

オプションのスクリーンショット機能を有効にすると、ディクテーション中にアクティブウィンドウの画像をキャプチャし、お客様が指定したディレクトリに保存します。スクリーンショットはローカルに保存され、外部サーバーには送信されません。この機能はデフォルトで無効です。

### クリップボード

KoeKey は認識結果をアクティブアプリに入力するため、システムクリップボードを使用してペースト（Ctrl+V / Cmd+V）を行います。デフォルトでは、入力後にクリップボードの元の内容が復元されます。設定を変更することで、認識結果をクリップボードに残すこともできます。

### アプリ内購入

KoeKey はプラットフォームのネイティブストア（Windows では Microsoft Store、macOS では App Store）をアプリ内購入に使用します。購入処理はすべてプラットフォーム側で処理されます。KoeKey は決済情報を収集・処理しません。

### テレメトリ・アナリティクス

KoeKey はテレメトリ、アナリティクス、クラッシュレポート、使用状況データを**一切収集しません**。ご利用状況に関する情報が開発者や第三者に送信されることはありません。

## データの保護方法

- クラウドプロバイダーへの音声送信には TLS 暗号化を使用します。
- API キーは OS レベルのセキュア資格情報ストアに保存され、キャッシュは自動で期限切れになります。
- 文字起こし履歴は AES-256-GCM で暗号化されます。
- KoeKey が管理する外部サーバーにデータは保存されません。
- KoeKey はお客様のデータを保護するため、現在の技術水準に沿った上記の措置を講じています。ただし、いかなるソフトウェアも絶対的なセキュリティを保証することはできない点をご了承ください。

## 第三者サービス

KoeKey はご利用状況に応じて以下の第三者サービスに接続します。

### クラウド音声認識（ユーザー操作による）

クラウドプロバイダーを設定・使用すると、音声データが選択したプロバイダーに送信されます:

- **Deepgram**: [プライバシーポリシー](https://deepgram.com/privacy)
- **Soniox**: [プライバシーポリシー](https://soniox.com/policies/privacy-policy)

各プロバイダーは独自のプライバシーポリシーに基づきデータを処理します。KoeKey が送信するのは音声データと API キーのみで、その他の個人情報は送信されません。

### モデルのダウンロード（ユーザー操作による）

ローカル処理を選択しモデルをダウンロードする場合、KoeKey は **Hugging Face**（`huggingface.co`）または **GitHub**（`github.com`）からモデルファイルをダウンロードします。個人データは送信されませんが、ダウンロード中にお客様の IP アドレスがホスティングサービスに送信されます。

- **Hugging Face**: [プライバシーポリシー](https://huggingface.co/privacy)
- **GitHub**: [プライバシーに関する声明](https://docs.github.com/ja/site-policy/privacy-policies/github-general-privacy-statement)

### プラットフォームストア

アプリ内購入の検証はプラットフォームのネイティブストア（Microsoft Store または App Store）が行います。KoeKey は決済データを直接処理しません。

## お客様の管理手段

お客様はデータを完全に管理できます:

- **ローカル処理の選択**: ローカル Whisper モデルを使用することで、すべての音声データをデバイス上に保持できます。
- **クラウドプロバイダーの選択**: 使用するプロバイダーはお客様が決定し、ご自身の API キーを設定します。
- **文字起こし履歴の削除**: アプリ内からいつでも履歴を消去できます。
- **API キーの削除**: アプリの設定から保存された API キーを削除できます。
- **出力ファイルの管理**: ディスクに保存したファイルはファイルマネージャーから削除できます。

## データの削除

- **文字起こし履歴・設定**: ローカルのアプリデータディレクトリに保存されます。アプリ内から履歴を削除するか、ディレクトリを手動で削除できます。
- **API キー・暗号化キー**: OS の資格情報ストアに保存されます。アプリの設定から削除できます。アプリのアンインストールでは資格情報ストアのエントリが自動削除されない場合があります。Windows の資格情報マネージャーまたは macOS のキーチェーンアクセスから手動で削除できます。
- **出力ファイル**: お客様が選択した場所に保存されており、手動での削除が必要です。

## 準拠法

KoeKey は適用されるプライバシー法令への準拠を目指して設計されています。GDPR、CCPA およびこれに類する法令のもと、お客様は個人データへのアクセス、訂正、削除の権利を有します。すべてのデータはお使いのデバイスにローカル保存されるため、これらの権利はお客様自身で直接行使できます。

## お子様のプライバシー

KoeKey は 13 歳未満のお子様を対象としていません。お子様の個人情報を意図的に収集することはありません。
